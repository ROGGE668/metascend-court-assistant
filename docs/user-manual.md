# 用户操作手册

## 适用设备

+ macOS 14+
+ Apple Silicon Mac（M1/M2/M3/M4/M5）
+ 16GB 内存及以上
+ 麦克风（内置或外接）
+ 耳机/蓝牙耳机（可选）

## 首次使用

1. 安装依赖：
   ```bash
   uv sync --extra dev
   ```
2. 下载模型（首次需联网）：
   ```bash
   export HF_ENDPOINT=https://hf-mirror.com   # 国内建议设置
   uv run python scripts/download_models.py
   ```
3. 启动应用：
   ```bash
   uv run python src/pipeline.py
   ```

## 开庭前准备

1. 选择案由：借贷 / 离婚 / 劳动 / 合同。
2. 在“庭审实时辅助”页完成角色声纹校准。
3. 连接耳机，测试音量。
4. 根据需要开启或关闭录音模式。

## 庭审中

+ 应用自动监听并转写对话，庭审实时辅助页面实时更新。
+ 对方发言触发法律策略时，页面会显示法律提示和应对建议。
+ 耳机播报策略（若开启 TTS）。
+ 按 `q`、`Esc` 或点击窗口关闭按钮退出。

## 一键静音

+ 快捷键：`Space` 或耳机触控。
+ 静音后状态灯变红，所有 TTS 停止，UI 仍继续转写。
+ 再次按快捷键恢复。

## 纯黑屏模式

+ 在设置中开启“纯黑屏模式”。
+ 屏幕只保留一个小状态指示灯，降低被注意的概率。
+ 双击状态灯可临时显示字幕。

## 庭后查看

+ 切换到 Chat 页面查看历史会话。
+ 若开启加密录音/日志，可用密码解密本地数据。
+ 支持基于本地知识库继续分析。

## 常见问题

**Q: 转写延迟高怎么办？**
A: 检查 ASR 模型是否为 `large-v3-turbo`；关闭录音模式；确保安静环境。

**Q: 识别不准？**
A: 在 `.env` 中补充案件热词；靠近麦克风说话；避免多人同时说话。

**Q: 耳机声音被旁人听到？**
A: 降低音量；使用骨传导耳机；开启闪避功能。

**Q: 法庭不允许电子设备？**
A: 请遵守法庭规定。本产品提供庭前模拟训练功能，可在庭前准备阶段使用。
